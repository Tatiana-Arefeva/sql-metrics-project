# SQL-проект: расчёт метрик сервиса доставки еды

## 📌 Описание
Проект выполнен для сервиса доставки еды «Всё.из.кафе». Подготовлены данные для дашборда ключевых бизнес-метрик по городу Оренбург за период май — июль 2025.

## 📊 Рассчитанные метрики

| Метрика | Описание |
|---------|----------|
| **DAU** | Ежедневные активные пользователи (событие `order`) |
| **Conversion Rate** | Конверсия в заказ |
| **Средний чек** | Средняя сумма комиссии на заказ (по месяцам) |
| **LTV ресторанов** | Топ-3 ресторана по совокупной комиссии |
| **LTV блюд** | Топ-5 блюд в топ-2 ресторанах |
| **Retention Rate** | Удержание пользователей (дни 1–7) |
| **Retention по месяцам** | Удержение в разрезе когорт (май / июнь / июль) |

## 🛠 Использованные приёмы SQL
- `JOIN` (объединение таблиц)
- `WITH` (CTE — общие табличные выражения)
- `DATE_TRUNC`, `CAST` (работа с датами)
- `SUM`, `COUNT`, `AVG`, `FILTER`
- Оконная функция `OVER` для расчёта retention_rate
- `LIMIT` для топ-3 и топ-5
