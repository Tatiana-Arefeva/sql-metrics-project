## 🔍 О проекте

Проект посвящён анализу ключевых метрик сервиса доставки еды с использованием SQL и визуализации в DataLens.

Цель — оценить поведение пользователей и эффективность бизнеса.

---

## 🎯 Бизнес-задача

- Проанализировать динамику заказов
- Оценить средний чек
- Изучить поведение пользователей
- Найти точки роста продукта

---

## 📁 Данные

Датасет включает:

- заказы
- пользователей
- время заказов
- стоимость заказов

---

## 📊 Рассчитанные метрики

| Метрика | Описание |
|---------|----------|
| **DAU** | Ежедневные активные пользователи (событие `order`) |
| **Conversion Rate** | Конверсия в заказ |
| **Средний чек** | Средняя сумма комиссии на заказ (по месяцам) |
| **LTV ресторанов** | Топ-3 ресторана по совокупной комиссии |
| **LTV блюд** | Топ-5 блюд в топ-2 ресторанах |
| **Retention Rate** | Удержание пользователей (дни 1–7) |
| **Retention по месяцам** | Удержение в разрезе когорт (май / июнь / июль) |

---

## 🧠 Ключевые инсайты

- Обнаружены периоды снижения активности пользователей
- Поведение пользователей отличается по дням недели
- Средний чек варьируется в зависимости от времени

---

## ⚠️ Проблемы

- Нестабильный спрос
- Возможное снижение пользовательской активности
- Потенциальные точки оттока клиентов

---

## 💡 Рекомендации

- Запуск акций в периоды снижения спроса
- Улучшение удержания пользователей (retention)
- Дополнительный анализ пользовательских сегментов
